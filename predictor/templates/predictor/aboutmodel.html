About the Model

Model Architecture

Our model is a Convolutional Neural Network (CNN) with 5 convolutional blocks and dropout layers to prevent overfitting.

Performance Metrics

The model achieves high accuracy on the EuroSAT dataset, making it effective for land cover classification tasks.

Usage

This model is designed for classifying land use from Sentinel-2 imagery, leveraging advanced image processing techniques to yield precise results.
